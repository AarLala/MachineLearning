# Major Machine Learning Papers Implementations

This repository contains my implementations of major Machine Learning papers.  

## Getting Started

To run the code, clone this repository to your local machine and follow the steps below.

### Windows

1. **Create a virtual environment**:

    ```bash
    python -m venv venv
    ```

2. **Activate the virtual environment**:

    ```bash
    venv\Scripts\activate
    ```

3. **Install dependencies**:

    ```bash
    pip install numpy
    ```
